# Security Policy

## Reporting a vulnerability

Please **do not** report security vulnerabilities via public GitHub Issues in this repository.

Instead, report potential security issues privately to:

- Email: support@adiscon.com

When possible, include:

- affected product and version
- affected component or feature
- impact and attack scenario
- steps to reproduce (or a minimal proof of concept)
- any relevant logs, crash dumps, or configuration details (redacted as needed)

## Scope

This repository is a community collaboration space and does not contain Adiscon product source code. Security reports should focus on released Adiscon products and services rather than repository content.

## Coordinated disclosure

We appreciate responsible disclosure. Please avoid publishing details until a fix or mitigation is available, and do not test in ways that could impact other users or systems without explicit permission.
