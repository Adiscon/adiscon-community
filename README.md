# Adiscon Community

This repository is the public collaboration space for selected Adiscon products: a place for feature requests, roadmap discussion, and UI text customization resources (including language files).

Useful starting points:

- Adiscon: https://www.adiscon.com/
- rsyslog: https://www.rsyslog.com/
- WinSyslog: https://www.winsyslog.com/
- EventReporter: https://www.eventreporter.com/
- MonitorWare Agent: https://www.mwagent.com/

## What this repo is for

- Feature requests and UI/UX improvement ideas (via GitHub Issues)
- Roadmap discussion and prioritization signals
- UI text customization, language file sharing, and wording improvements

## Products covered

- WinSyslog (Windows syslog server; first syslog server for the Windows platform, released in 1996)
- EventReporter (Windows Event Log forwarding; first Windows Event Log-to-syslog forwarder, created in 1997)
- MonitorWare Agent (actively developed Windows monitoring + log processing suite)
- rsyslog Windows Agent (Windows companion for rsyslog users)
- Adiscon LogAnalyzer (web viewer for syslog and event data; GPL)

## The rsyslog connection

Adiscon sponsors and maintains the rsyslog project. If you run rsyslog on Linux/Unix and need Windows-side log collection and forwarding, the products above are designed to work well in mixed environments.

## What this repo is not

- This is not a product source code repository. Do not submit product source code here.
- This is not an official support channel. For product support, licensing, or operational issues, please use Adiscon’s support channels.

## Contributing

Please keep contributions within scope: feature requests, roadmap discussion, UI customization resources, and language file improvements.

For language files, do not change keys or file structure. Only propose improvements to translated text values and preserve formatting.
